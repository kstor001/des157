January 31, 2017

I really enjoy dribbble.com's user interface. The design is very clean and straight forward with a clear hierarchy. A user can easily navigate the page and dribbble offers hover features for the dropdown menu at the top and for each post giving a detailed description of the design entry. The interface uses primarily grays and white with one pink spot color for the "sign-up" button. This works well to allow the design entries to be the main display of color, attracting the user's interest. The interface is very clear and organized, with all information contained clearly to each section.The website appears simple and shows good restraint of design, allowing the user's eye to flow naturally to the work that catches their eye best by aesthetic preference.
