User Needs

  • User needs a way to navigate the application
  • User needs clear instructions for how to navigate application
  • User needs to be informed of the purpose of the application
  • User needs to be informed of the benefits of using the application
  • User needs to feel empowered from using the application
  • User needs a clear presentation of information
  • User needs interactive features to improve aesthetic enjoyment
  • User needs a solidified color palette for the application, too many colors makes the user not want to use the application
  • User needs a way to feel connected whether it be to themselves or others
  • User needs their information to be stored so that they can look back and grow from the past
